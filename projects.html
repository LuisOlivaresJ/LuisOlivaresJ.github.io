An open source application to perform radiochromic film dosimetry.

Dosepy is intended to be an accessible tool for medical physicist in radiation oncology to perform patient-specific dose verification based on film measurements.

The software works with scanned films (in TIFF format) and a DICOM file (typically exported from a treatment planning system). A 2D gamma analysis can be performed in order to evaluate the similarity between the measured (scanned film) and the planed (DICOM file) dose distributions.

Pyportal

Pyportal is an app for performing TG-307 quality assurance (QA) tests for routine QA of (R) Varian EPID systems.

Features

  • SQL database storage for constancy analysis.
  • One-click fast test assessment.

Relative dose 1d

Python package to read 1-dimensional dose profile from a text file to perform subtraction and gamma analysis.
